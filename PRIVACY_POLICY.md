## Repo2Prompt Privacy Policy

Repo2Prompt is a macOS app that turns a folder you choose into a structured prompt.

### Data We Process

- Folder and file contents from locations you explicitly choose in the app
- Recent folder names, paths, and security-scoped bookmark data stored locally on your Mac
- App preferences stored locally on your Mac

### How We Use Data

- To scan the selected folder and calculate token estimates
- To let you choose which files and folders are included in the generated prompt
- To reopen folders you previously selected
- To copy the generated prompt to your clipboard when you request it

### Data Sharing

- Repo2Prompt does not create user accounts
- Repo2Prompt does not sell personal information
- Repo2Prompt does not send repository contents to the developer
- Repo2Prompt does not include third-party analytics

### Storage

- Recent folder metadata and preferences are stored locally using Apple system APIs
- Repository contents are processed locally on-device by the app

### User Control

- You choose which folder to open
- You choose which files and folders are selected
- You can remove recent folders from the app at any time
